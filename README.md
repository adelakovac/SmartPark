# SmartPark 🚗

SmartPark is a web application developed as part of a Senior Design Project.

## 📌 Description
SmartPark helps users find, manage, and reserve parking spots efficiently.

## 🎯 Problem
Parking management in urban areas is inefficient and lacks real-time tracking.

## 💡 Solution
SmartPark allows users to:
- view parking locations
- generate parking spots
- reserve available spots
- track availability in real-time

## 🚀 Features
- User authentication
- Parking location management
- Parking spot generation
- Reservation system
- Dashboard statistics
- Search & filtering

## 🛠 Tech Stack
- Laravel (PHP)
- MySQL
- Blade
- Tailwind CSS
